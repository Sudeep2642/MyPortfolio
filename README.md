# 🚀 My Portfolio

Welcome to the repository for my professional portfolio website! This project serves as a central hub to showcase my technical expertise, the projects I've built, and how to get in touch with me.

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** CSS3 / Tailwind CSS (Optional: change if you're using something else)
- **Icons:** SVG-based custom icons
- **Deployment:** [Vercel / GitHub Pages / Netlify]

## ✨ Key Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Project Gallery:** A curated list of my best work with links to live demos and source code.
- **Contact Form:** A way for recruiters and collaborators to reach out directly.
- **Dynamic Assets:** Uses lightweight SVG graphics for fast loading times.
